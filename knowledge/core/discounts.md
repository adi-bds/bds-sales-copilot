# BDS Active Discount Codes

**Source:** Shopify discounts export — active codes only (test codes excluded)
**Rule:** Only share a discount code when a client explicitly asks for one, requests a price match, or pushes back on price. Never suggest proactively.

---

## Usage Guidelines

- **Always check client tier first** (if in callprep): Tier 1 VIP = up to 10%; Tier 2 Key = 7–8%; Tier 3 Active = 5–7%
- **Never stack** — none of these codes combine with other discounts
- **LP100 is restricted** — 100% off, only 2 uses remaining. Do not share without manager approval.
- **Dated single-use codes** (e.g. RET8, RET12, WB15, REPRINT15 with a date in the name) are generated for specific customers — do not give these out to anyone unless they were specifically issued for that client.

## How to Offer a Discount

When a client pushes back on price, suggest the lowest appropriate code first:

1. New client → FIRST7 (7%)
2. Repeat client → TAKE10 (10%) or your rep-specific code
3. Client threatening to leave → RET8 or RET12 (escalate to manager if RET12)
4. Lapsed client returning → WB15
5. FIFA order → FIFA7

Don't volunteer the percentage ladder. Offer one code, see if they accept.
